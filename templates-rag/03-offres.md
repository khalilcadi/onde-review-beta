# BLOC 3 - OFFRE PRODUIT

> **bloc_id** : `bloc_3`
> **Utilise par** : Prospection, Conversationnel
> **Priorite** : #4 - Ce qu'on vend concretement

---

## A quoi sert ce bloc ?

Ce bloc decrit ton offre en detail : ce qu'elle inclut, ce qu'elle promet, comment elle fonctionne. L'IA l'utilise pour mentionner les bons arguments dans les messages et repondre aux questions sur l'offre.

---

## Sections a remplir

### 1. Positionnement de l'offre

> Une definition claire et assumee de ce que tu vends.

```
[A REMPLIR]

Questions guides :
- Nom de l'offre ?
- Categorie ? (SaaS, service, formation, accompagnement, outil...)
- En une phrase : c'est quoi exactement ?
```

### 2. Promesse centrale

> Le resultat concret que ton client obtient en achetant.

```
[A REMPLIR]

Questions guides :
- Quel resultat mesurable / tangible ?
- En combien de temps les premiers resultats ?
- Qu'est-ce qui change concretement dans le quotidien du client ?
```

### 3. Ce que l'offre inclut (le contenu)

> Liste detaillee de ce qui est inclus.

```
[A REMPLIR]

Inclus :
- ...
- ...
- ...

Non inclus (pour eviter les malentendus) :
- ...
- ...
```

### 4. Comment ca marche (le processus)

> Les etapes que le client traverse apres l'achat.

```
[A REMPLIR]

Etape 1 : ...
Etape 2 : ...
Etape 3 : ...
Etape 4 : ...

Questions guides :
- Quelle est l'experience post-achat ?
- Combien de temps pour etre operationnel ?
- Quel effort demande au client ?
```

### 5. Pour qui c'est fait (et pas fait)

> Relie a l'ICP mais specifique a cette offre.

```
[A REMPLIR]

C'est fait pour :
- ...
- ...

Ce n'est PAS fait pour :
- ...
- ...
```

### 6. Differenciateurs cles

> Ce qui rend ton offre unique vs les alternatives.

```
[A REMPLIR]

Questions guides :
- Pourquoi toi et pas un concurrent ?
- Quel est ton "unfair advantage" ?
- Qu'est-ce que les autres ne font pas / mal ?
```

### 7. Preuves et resultats

> Temoignages, chiffres, cas clients qui soutiennent la promesse.

```
[A REMPLIR]

Questions guides :
- Temoignages clients marquants ?
- Chiffres de resultats (avant/apres) ?
- Nombre de clients / utilisateurs ?
- Prix, certifications, reconnaissances ?
```

### 8. Prix et modele economique

> Structure tarifaire (si tu veux que l'IA puisse en parler).

```
[A REMPLIR]

Questions guides :
- Quel est le prix ?
- Modele : abonnement, one-shot, freemium, sur devis ?
- Y a-t-il des frais de setup / engagement ?
- Existe-t-il une offre d'essai ou garantie ?
```

Jarvis centralise, analyse et orchestre ton business au quotidien
