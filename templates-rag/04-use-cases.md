# BLOC 4 - USE CASES

> **bloc_id** : `bloc_4`
> **Utilise par** : Prospection, Conversationnel

---

## A quoi sert ce bloc ?

Ce bloc catalogue les situations concretes ou ton produit/service aide le client. L'IA l'utilise pour adapter le message au contexte specifique du lead (ex: "je vois que vous etes en phase de croissance, voici comment on peut aider").

---

## Sections a remplir

### 1. Principe fondamental

> Comment ton produit decide quel use case activer pour un client.

```
[A REMPLIR]

Questions guides :
- Ton produit fait-il toujours la meme chose ou s'adapte-t-il au contexte ?
- Quels criteres determinent quelle fonctionnalite / approche proposer ?
- Y a-t-il des situations ou tu refuses d'intervenir ?
```

## Exemple de section bien redigee

```
### Use Case #1 : Pilotage quotidien

Situation client : Solopreneur en phase stable, CA regulier mais sans visibilite
Probleme concret : Il prend ses decisions "au feeling", sans donnees centralisees
Ce que Jarvis fait :
- Consolide les KPIs business chaque matin
- Identifie les actions prioritaires du jour
- Alerte sur les signaux faibles (lead chaud, relance oubliee)
Resultat obtenu : Le client passe de 0 visibilite a un cockpit clair en 5 min/jour
Exemple reel : "Avant Jarvis, je decouvrai mes problemes trop tard. Maintenant je les vois arriver."
```
