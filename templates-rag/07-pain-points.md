# BLOC 7 - PAIN POINTS

> **bloc_id** : `bloc_7`
> **Utilise par** : Prospection, Scoring, Conversationnel
> **Priorite** : #3 - Les douleurs guident tout le messaging

---

## A quoi sert ce bloc ?

Ce bloc cartographie les douleurs reelles de tes clients. C'est la fondation du messaging : l'IA utilise ces douleurs pour creer de la resonance dans les messages de prospection. Un message qui touche un pain reel convertit 10x mieux qu'un message generique.

**Regle cle : un prospect n'a jamais "tous" les problemes. Il a un pain dominant.**

## Exemple de section bien redigee

```
### Famille de pain #1 : Surcharge mentale

Nom : Surcharge mentale (ou "charge invisible")
Description : Le solopreneur porte tout seul. Pas de DirCo, pas de CMO,
pas de bras droit. Chaque decision repose sur lui.

Symptomes visibles :
- Procrastination sur les taches strategiques
- Reponses en retard aux clients
- Prospection irreguliere, "quand j'ai le temps"
- Decisions prises a l'instinct plutot qu'aux donnees

Ce que le prospect dit :
- "Je manque de temps"
- "J'ai trop de trucs en meme temps"
- "Je sais ce que je devrais faire, mais..."

Ce que le prospect ressent :
- Culpabilite de ne pas avancer assez vite
- Sentiment d'etre seul face a tout
- Peur que ca finisse par craquer

Comment Jarvis repond :
- Prend en charge la charge decisionnelle quotidienne
- Priorise a sa place (1 action claire vs 10 options floues)
- Reduit le bruit mental en centralisant l'info
```
