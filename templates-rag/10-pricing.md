# BLOC 10 - STRATEGIE PRICING

> **bloc_id** : `bloc_10`
> **Utilise par** : Conversationnel
> **Priorite** : #9 - Strategie de prix

---

## A quoi sert ce bloc ?

Ce bloc donne a l'IA la logique derriere ton pricing pour qu'elle puisse en parler naturellement en conversation. L'objectif n'est pas de "closer" mais de maintenir une posture claire et coherente sur le prix.

---

## Sections a remplir

### 1. Principe fondateur du pricing

> La philosophie derriere ton prix.

```
[A REMPLIR]

Questions guides :
- Ton prix reflete quoi exactement ? (valeur, temps, resultat, acces...)
- Est-ce un pricing par feature, par valeur, par utilisateur ?
- Quelle est ta position : premium, accessible, mid-range ?
```

### 2. Structure tarifaire

> Les chiffres concrets.

```
[A REMPLIR]

Offre principale :
- Nom : ...
- Prix : ...
- Frequence : (mensuel, annuel, one-shot)
- Ce qui est inclus : ...

Offre(s) secondaire(s) (si applicable) :
- ...

Frais additionnels (setup, options...) :
- ...

Engagement minimum :
- ...
```

### 3. Ancrage de valeur

> Comment justifier le prix sans se justifier.

```
[A REMPLIR]

Questions guides :
- Combien coute l'alternative humaine ? (embauche, freelance, agence)
- Combien le prospect perd en n'ayant pas ta solution ? (temps, CA, opportunites)
- Quel est le ROI type d'un client ? (en combien de temps il se rembourse)
- Y a-t-il un chiffre d'ancrage puissant ? ("au lieu de X, tu paies Y")
```

### 4. Objections prix et reponses

> Les objections specifiques au prix (peut renvoyer au Bloc 5 pour le detail).

```
[A REMPLIR]

Objections courantes :
- "C'est trop cher" -> ...
- "Pas de budget" -> ...
- "Il y a moins cher" -> ...
- "Je ne vois pas le ROI" -> ...
```

### 5. Regles de negociation

> Ce que l'IA peut ou ne peut pas faire sur le prix.

```
[A REMPLIR]

Questions guides :
- L'IA peut-elle proposer une remise ? Si oui, dans quelles conditions ?
- Y a-t-il une offre d'essai / garantie satisfait ?
- Jusqu'ou tu es pret a aller en negociation ?
- Y a-t-il des cas ou tu offres le service ? (beta testeurs, partenaires...)
```

### 6. Evolution pricing prevue

> Si ton prix va changer, l'IA doit le savoir.

```
[A REMPLIR]

Questions guides :
- Le prix actuel est-il un prix de lancement / early access ?
- Quand prevois-tu une hausse ?
- Y a-t-il un "grandfather" pour les early adopters ?
```

---

## Exemple de section bien redigee

```
### Ancrage de valeur

Jarvis n'est pas facture pour ce qu'il execute,
mais pour la clarte, la structure et les decisions qu'il apporte.

Ancrage :
- Un business partner humain = ~2 000 EUR/mois minimum
- Un commercial junior = ~2 500 EUR/mois charges
- Jarvis Start = 79 EUR/mois + 500 EUR setup

ROI type :
- 1 client gagne grace a la prospection structuree = 500-5000 EUR
- Payback period : 1er mois dans 80% des cas
```
